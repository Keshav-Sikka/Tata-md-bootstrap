# Tata MD Healthcare Landing Page (Alpha)

A pixel-focused recreation of a healthcare landing page built as part of my frontend internship training.

The objective of this project was to accurately translate a Sketch design into a responsive Bootstrap-based webpage while becoming familiar with Bootstrap's layout system, components, and utility classes.

> **Current Status:** Alpha Release

## Project Description

This project recreates the landing page of a healthcare platform using HTML and Bootstrap 5.

The emphasis was on understanding modern page layouts, Bootstrap's grid system, responsive containers, cards, navigation bars, positioning utilities, and component composition rather than implementing application logic.

---

# Internship Constraints

This project was completed under the following constraints:

- **Bootstrap only** for styling and layout.
- **No custom CSS files** were allowed.
- **No JavaScript** was allowed.
- Inline styles were used only where Bootstrap utilities were insufficient to reproduce the design accurately.

These constraints were intentionally imposed as part of the learning process to build proficiency with Bootstrap before introducing custom CSS and JavaScript.

Planned improvements include:

- Deploy the project for live viewing.

> JavaScript functionality is **not** planned for this project. Rather than turning this static design into an application, future JavaScript work will be invested in projects where functionality and problem-solving are the primary focus.
